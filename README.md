# 🚌 Bus Booking System

## 📌 Description

This project is a Java-based Bus Booking System that allows users to search buses, book tickets, and manage reservations efficiently.

## 🚀 Features

* View available buses
* Book tickets
* Cancel bookings
* Simple and user-friendly interface

## 🛠️ Technologies Used

* Java
* Swing (GUI)
* MySQL (if used)

## 📂 Project Structure

* src/ → Contains all Java source code
* database/ → Contains DB files (if any)

## ▶️ How to Run

1. Open project in Eclipse or IntelliJ
2. Run the main class
3. Use the interface
